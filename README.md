# Halo-Code-Challenge

## How to Run
  * Using the Terminal, run `pip install -r requirements.txt` from root of this repository
  * Use the Command `python app.py` to get the application running on Port 3000
  * In a web browser, go to `http://localhost:3000/`

## Description
